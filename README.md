# carver_sql_to_es_api

#### 一：项目介绍
 **以下是通过SQL语句调取该接口的返回ESDML的一个后端接口项目** 

#### 二：项目结构
 **该项目没有使用go的主流框架做支撑，仅仅使用go的原生结构做的接口** 

#### 三：安装教程

1.  git clone 项目地址    【克隆项目到本地】
2.  go mod tidy          【需要重新加载一些第三方的依赖库】
3.  go build main.go     【重新打包成可执行文件main】
